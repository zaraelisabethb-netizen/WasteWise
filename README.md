# WasteWise

**AI-powered food waste prediction tool that helps identify foods most likely to be wasted.**

## Live Demo

https://f05fcbd5-11d3-43ff-8ce4-dd6144b8b3ab-00-2i41an95bdo6r.riker.replit.dev/

## What It Does

WasteWise is an AI application designed to predict when food is likely to be wasted. By analyzing information from food-waste-related sources and user inputs, it identifies foods with a higher risk of being thrown away. The goal is to help individuals and households make smarter purchasing, storage, and consumption decisions to reduce food waste.
